## ssm-core食用指南 ##

### 使用本工程前提： ###

- JAVA环境、MAVEN环境、GIT环境、MYSQL安装完成
- maven使用阿里云镜像

&nbsp;&nbsp;1.将工程从git上clone下来：

![](https://i.imgur.com/x5NQfIN.png)

&nbsp;&nbsp;3.修改jdbc参数，包括自己机器上的数据库名称以及用户名和密码

![](https://i.imgur.com/cTXpBmz.png)

&nbsp;&nbsp;4.确保mysql服务已经启动
